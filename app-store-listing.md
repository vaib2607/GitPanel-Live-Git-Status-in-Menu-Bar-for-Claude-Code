## Description (4000 chars max)

GitPanel gives you instant visibility into your git repositories without leaving your menu bar.

**Real-time repository status**
GitPanel watches your git index using macOS FSEvents — changes appear instantly when you stage, commit, or modify files. No polling, no delays. The 300ms debounce keeps the UI smooth during batch operations.

**Diff summary at a glance**
See exactly what changed: lines added and deleted, staged vs unstaged files, untracked files, and merge conflicts — all in one clean card. No need to run `git status` in a terminal.

**Branch switching**
Searchable branch list with create-new-branch support. Right-click any branch to checkout, copy its name, or delete it. See ahead/behind badges so you know exactly where you stand with remote.

**Commit and push**
Three separate actions: Commit staged changes, Push existing commits, or Commit & Push in one step. Keyboard shortcuts for power users: Cmd+Enter to commit, Cmd+Shift+Enter to commit and push.

**PR status**
Shows your open pull requests with title, author, and review status — powered by the GitHub CLI (`gh`). Distinct feedback when `gh` isn't installed vs when there are no PRs.

**Repo state detection**
GitPanel knows when you're in a rebase, cherry-pick, revert, bisect, or merge conflict. The menu bar icon changes to reflect the current state so you always know what's happening.

**Dynamic menu bar icon**
The icon changes based on your repo: clean (checkmark), dirty (up arrow), conflict (exclamation), rebase (branch icon). Works with both light and dark mode.

**Claude Code usage tracking**
See your real token counts and cost from Claude Code offline logs. Best-effort Cursor plan tier detection. Everything stays local — no data leaves your machine.

**Drag & drop staging**
Drag files from Finder directly onto the panel to stage them. The blue highlight border confirms the drop target.

**Keyboard shortcuts**
- Cmd+R: Refresh
- Cmd+Enter: Commit
- Cmd+Shift+Enter: Commit & Push

**Privacy-first**
GitPanel runs entirely on your Mac. Git operations use the local `git` CLI. Usage data is parsed from local logs. No analytics, no tracking, no network calls. Your code never leaves your machine.

**Open source**
GitPanel is free and open source under the MIT license. View the code on GitHub.
